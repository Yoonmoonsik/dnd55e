# Translation notes — French localization

Decisions and arbitration record for the `french.xml` update (coverage
aligned with `english.xml`). The glossary extracted from the corpus lives in
`tools/localization/glossary-french.json` (generated by `tools/localization/build-glossary.mjs`); the
tooling used throughout is in `tools/localization/` (`loca-diff`, `loca-apply`,
`loca-update`, `loca-validate`).

## Terminology priority rule

For every game term, the translation is chosen in this order:

1. **BG3 base game** — ground truth: the official French localization
   extracted from `French.pak` (218,261 entries, current patch);
2. **Official French D&D translation** (2024 preferred, via AideDD);
3. **Existing `french.xml` corpus** (majority variant when in conflict);
4. **New translation** as a last resort (always recorded here, with
   alternatives).

This priority applies to new entries AND retroactively to divergent existing
entries (see “Corpus-wide harmonization” below). Never translate a game term
from memory: grep the extracted BG3 localization first, then AideDD — the web
already got it wrong once (“Aspersion d'acide” reported by several sites; the
game files say “Aspersion acide”).

## User-arbitrated decisions

- **Warlock → Occultiste** (never “Sorcier-Lié”/“Sorcier”).
- **Cantrip → sort mineur** (BG3 term; replaces the corpus majority
  “tour de magie”).
- **Expertise → Expertise** (never “Maîtrise approfondie”).
- **Guidance → Assistance**, **Mage Slayer → Fléau des mages** (BG3 names).
- **Weapon Mastery**: “Maîtrise des armes” as the feature name,
  “maîtrise d'arme” inline — context-dependent.
- **Breath Weapon**: two distinct features keep their two established
  translations — “Arme de souffle” (Dragonborn trait) and “arme du souffle”
  (Druid Dragon Shape attack).
- **Distances in metres** (see “Metric conversion” below).
- Named by the user: **Extension d'ombre** (Lengthen Shadow), **Disparition
  dans les ténèbres** (Fade to Black), **Canalisateur** (Channeler),
  **Murmures des esprits** (Guiding Whispers), **Acier-Saints** (Steel
  Saints).

## Final validation and consistency passes

`tools/localization/loca-validate.mjs` checks, for all 4,683 entries: identical counts,
no duplicate uid, no orphan, versions aligned, identical inline structure
(LSTag by Tooltip, `<br>`, entities, placeholders [1]–[9], paragraph count),
and identical dice (XdY) and number multisets (with FR metric distances
normalized back to feet before comparison).

The consistency pass caught and fixed: 7 truncated domain/patron spell lists
(the translated boilerplate hid the missing list — found by the number
comparison), 2 lost LSTag tags, 3 missing paragraphs, « Gisombre »
(Shadowfell, BG3) replacing « Féeombre », « Restauration partielle » and
« Rayon affaiblissant » replacing older variants. Kept as-is: the double
space in « Désactiver  " » (identical in EN). EN typos were not reproduced
(“Thron Armor” → « Armure d'épines »).

Final state: **FR = EN = 4,683 entries; 0 missing, 0 stale, 0 orphan,
0 duplicate; 0 structural, dice or number mismatch; well-formed XML; entry
order, EOF and BOM identical to the original.**

## Metric conversion (user decision)

BG3 is natively metric in every language (“Move the target 3m closer” →
« de 3 mètres »); the official French D&D translation is metric as well; only
the mod's EN (book text) and the old corpus used feet. Per the priority rule,
**all distances were converted to metres**: 391 entries, 502 occurrences,
strict ×0.3 table (any value outside the table aborts the batch):
1 ft → 30 cm; 2 → 60 cm; 5 → 1,50 mètre (singular, official-French format);
10 → 3 m; 15 → 4,50 m; 20 → 6 m; 30 → 9 m; 40 → 12 m; 45 → 13,50 m;
60 → 18 m; 90 → 27 m; 120 → 36 m — written out as « mètres ». Agreement
adjusted (« 1,50 mètre supplémentaire »). The pre-existing « 24 mètres »
(superior darkvision; EN literally says “a distance of 24.”) is now
consistent with the rest. The validator's number check normalizes metres
back to feet, so the EN/FR number guarantee still holds.
